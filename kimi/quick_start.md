# AI Chat Client - 快速开始指南

## 🚀 立即开始使用

### 1. 安装依赖
```bash
npm install
```

### 2. 启动应用
```bash
npm start
```

### 3. 配置 API
1. 首次启动后，点击右下角的"设置"按钮
2. 输入您的 OpenAI API Key
3. 点击"保存设置"

### 4. 开始对话
1. 在主界面输入您的问题
2. 按 Enter 或点击发送按钮
3. 享受 AI 对话体验！

## 📋 功能亮点

✅ **多模型支持** - GPT-3.5 Turbo、GPT-4、GPT-4 Turbo  
✅ **流式响应** - 实时显示 AI 回复  
✅ **会话管理** - 创建新对话、清空历史  
✅ **参数配置** - 自定义模型参数  
✅ **现代化 UI** - 类似 Cherry Studio 的设计  
✅ **本地存储** - 设置自动保存  

## 🔧 配置说明

### 获取 API Key
1. 访问 [OpenAI 平台](https://platform.openai.com/)
2. 注册账号并创建 API Key
3. 在应用设置中输入 API Key

### 支持的模型
- GPT-3.5 Turbo (推荐，速度快)
- GPT-4 (功能强大，需要 GPT-4 权限)
- GPT-4 Turbo (最新版本)

### 可调整参数
- **温度**: 控制回复随机性 (0-2)
- **最大令牌数**: 限制回复长度
- **API URL**: 支持自定义 API 端点

## 🎨 界面特色

- **侧边栏**: 会话管理和设置入口
- **主界面**: 清晰的聊天区域和输入框
- **设置面板**: 完整的配置选项
- **响应式设计**: 适配不同屏幕尺寸

## 💡 使用技巧

1. **快速开始**: 使用欢迎界面的快捷按钮
2. **键盘操作**: Enter 发送，Shift+Enter 换行
3. **中断生成**: 点击"停止生成"按钮
4. **多会话**: 创建多个独立对话
5. **参数调优**: 根据需要调整模型参数

## 🔒 隐私说明

- API Key 本地存储，不会上传到任何服务器
- 对话内容仅用于与 AI 模型交互
- 所有数据都保存在本地
