# 模型管理功能测试指南

## 测试步骤

### 1. 打开模型管理
- 点击侧边栏底部的"模型管理"按钮（🤖图标）
- 应该看到模型管理模态框打开

### 2. 添加测试模型
- 点击"添加模型"按钮
- 填写以下信息：
  - 厂商：选择"OpenAI"
  - 模型名称：gpt-3.5-turbo
  - API Key：输入你的OpenAI API密钥（如果没有，可以使用测试密钥"sk-test"）
  - API地址：保持默认（https://api.openai.com/v1/chat/completions）
  - 最大令牌：2048
  - 温度：0.7
  - 启用：勾选
- 点击"保存模型"按钮

### 3. 测试模型连接
- 在模型列表中点击"测试"按钮
- 应该看到状态变为"在线"或"离线"

### 4. 在聊天中使用模型
- 关闭模型管理模态框
- 在主界面的模型选择器中选择刚添加的模型
- 在输入框中输入测试消息，如"你好"
- 点击发送按钮

### 5. 测试其他厂商（可选）
可以重复步骤2-4，测试其他厂商如Anthropic、Google等

## 预期结果
- 模型管理界面正常显示
- 能够成功添加、编辑、删除模型
- 模型连接测试功能正常工作
- 在聊天中能够成功切换和使用不同模型
- 错误处理机制正常工作（如无效的API密钥会显示错误信息）

## 注意事项
- 如果没有真实的API密钥，可以使用测试密钥，但连接测试会失败
- 某些厂商的API可能需要特定的网络环境
- 模型测试可能需要一些时间，请耐心等待
